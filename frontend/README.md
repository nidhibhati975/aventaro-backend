# Aventaro Frontend (Bare React Native)

This app runs on React Native CLI (no Expo).

## Setup

1. Install dependencies

```bash
npm install
```

2. Start Metro

```bash
npm run start
```

3. Run Android

```bash
npm run android
```

4. Run iOS (macOS only)

```bash
npm run ios
```

## Environment

The frontend and backend both read from the single root [`.env`](C:/Users/nidhi/Aventaro/.env).

Required frontend variables:

- `API_BASE_URL`
- `BACKEND_URL`
